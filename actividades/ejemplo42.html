En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproducción auomática y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo esta presente, activa esa opción. Si se omite el atributo, la opción está desactivada. Si el navegador noa admite el elemento <video> o el formato de video utilizado, se mostrará lo que esté entre la <video> de apertura y las etiquetas de <#47> de cierre.

preload: este atributo le indica al navegador qué hacer cuando se carga la página. Puede tener uno de los tres valores:
none: el navegador no debe cargar el video hasta que el usario presiona Play.
auto: el navegador debe descargar el video cuando se cargue la página.
metadata: el navegador debe simplemente recopilar la información como el tamaño, el primer fotograma, la lista de pistas y la duración.
